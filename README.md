WeRunThis
=========

Discover Amazing Runs.  Anywhere.

WeRunThis is a simple app that allows users to draw run routes on maps and share them.  The app is built with Backbone.js and uses the Google Maps API.

FRAMEWORKS USED:
- Backbone.js
- backbone-localStorage
- Google Maps API
- Bootstrap css
